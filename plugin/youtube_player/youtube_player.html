Plugin Krita - Lecteur Youtube

Ce plugin permet d'afficher des vidéos directement sur le canvas de Krita en utilisant un docker.

Comment ça marche ?

  • L'utilisateur fournit une URL d'une vidéo YouTube ou un autre lien vidéo compatible.
  • Le plugin télécharge la vidéo en utilisant yt_dlp.
  • Une fois téléchargée, la vidéo est découpée en images avec OpenCV.
  • Les images sont ensuite affichées une par une sur le canvas de Krita, créant un effet de lecture vidéo.

Prérequis

  • Avoir Krita installé.
  • Avoir ytb_dl et open-cv installer avec python : pip install yt-dlp opencv-python
